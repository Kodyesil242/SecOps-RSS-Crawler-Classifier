# QLineSec

Proje Amacı
Siber güvenlik dünyasındaki gelişmeleri (Exploitler, Yamalar, Haberler) RSS kaynaklarından toplayıp akıllıca sınıflandıran bir haber alma sistemi.

Beklenen Özellikler
Çoklu Kaynak: NVD, Exploit-DB, Security Blogları gibi kaynakların RSS beslemelerinin eklenmesi.
Akıllı Sınıflandırma: İçeriğe göre (Web, Network, Crypto vb.) otomatik etiketleme.
Puanlama (Ranking): Haberin önem derecesini (Impact) belirleyen bir algoritma (örn: CVE skoruna göre).
Doğrulama: Aynı haberin birden fazla kaynaktan teyit edilmesi (Cross-referencing) ile güven skoru oluşturma.
Teknik Odak
Yöntem: RSS Parsing, NLP (Basit metin işleme)
